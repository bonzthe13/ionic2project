The technical exam using cross platform on Ionic 2 and Angular 2 and in typescripting and this is for mobile application. The demoproject is all about creating simple user log in using email as username and has a password. and the objectives of the project is to validate the simple entries and it will be first loaded when the application run. the validation is made using method as a service function to control whole the validation process. next the application also has a capable of creating simple project name entry using local storage and has capability of checking the existing project name and has a listing of all existing project. when the user login validation is successfull the all project listing is displayed using ionic listing method.
